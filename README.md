# Hostel Management System

A full-stack hostel management application built with FastAPI, React, and SQLite for managing students, rooms, and hostel records.
## Tech Stack

### Backend

* FastAPI
* Python
* SQLite

### Frontend

* React
* JavaScript
* HTML
* CSS

## Features

* Student registration and management
* Room allocation and tracking
* Hostel record management
* User-friendly dashboard
* Fast API-based backend services
* Responsive React frontend


## Installation

### Clone the repository

```bash
git clone https://github.com/preshina/HostelManagementSystem.git
cd HostelManagementSystem
```

### Backend Setup

```bash
cd Backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend Setup

```bash
cd Frontend
npm install
npm start
```

## API Documentation

FastAPI automatically generates API documentation.

After starting the backend, visit:

```text
http://localhost:8000/docs
```

## Database

The application uses SQLite for data storage. SQLite was chosen for its simplicity and ease of deployment during development.
